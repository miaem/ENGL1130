The Book of Phoenix

About

The Book of Phoenix is a futuristic novel that is very geographically dependent. The goal of this project is to visualize this book on a map. This interactive map pinpoints important locations as well as the origin of multiple characters. Connections and relationships can be explored that would not be so apparent without mapping.
